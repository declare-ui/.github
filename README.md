# .github

Organization-wide community health files and templates for the DeclareUI ecosystem.

---

## What's here

| File / Folder | Purpose |
|:--------------|:--------|
| `profile/README.md` | Organization profile displayed on [github.com/declare-ui](https://github.com/declare-ui) |
| `CONTRIBUTING.md` | Contribution guidelines shared across all repos |
| `ISSUE_TEMPLATE/` | Issue templates (bug report, feature request) |
| `PULL_REQUEST_TEMPLATE.md` | PR template |
| `CODE_OF_CONDUCT.md` | Community code of conduct |
| `SECURITY.md` | Security policy and vulnerability reporting |

## How it works

GitHub automatically uses files from this `.github` repo as defaults for all repositories in the `declare-ui` organization that don't have their own versions.

## Ecosystem

See the full list of repositories at [github.com/declare-ui](https://github.com/declare-ui).
